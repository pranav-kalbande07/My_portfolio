Projects

Project One :

🍽️ MyCafeteria – Python-Based Cafeteria Management System

Description:
MyCafeteria is a simple yet efficient terminal-based Cafeteria Management System developed using pure Python. This project simulates real-world cafeteria operations by allowing users to place food orders, view the menu, calculate bills, and manage orders — all through a command-line interface.

Key Features:
📋 Dynamic Menu Display
🛒 Order Placement and Cart Management
💰 Bill Calculation with Taxes
🗑️ Order Cancellation and Reset
🔐 Basic Admin Controls (optional: for editing menu or viewing reports)

Technologies Used:
Python (Core Logic & Data Handling)
File Handling (to save menu or order data)

Purpose:
The project was built to strengthen core Python skills, including object-oriented programming, user interaction, and real-time logic handling, without relying on any external frameworks or libraries.

Project Two:

✊✋✌️ Stone Paper Scissor – Python Terminal Game

Description:
Stone Paper Scissor is a classic command-line game developed entirely using Python. This interactive project allows users to play the popular game against the computer, with outcomes determined through randomization and conditional logic. It offers a fun way to demonstrate decision-making, control flow, and user interaction in Python.

Key Features:
🎮 User vs Computer Gameplay
🤖 Computer move generated using random module
🧠 Winner determination using game rules logic
🔁 Play again feature with score tracking (optional)
🖥️ Simple and user-friendly terminal interface

Technologies Used:
Python (Core Game Logic)
random module (for computer choice generation)
Conditional Statements and Loops
String Formatting and User Input Handling

Project Three:

❌⭕ Tic Tac Toe – Python Terminal Game

Description:
Tic Tac Toe is a classic two-player game built using core Python, playable through the terminal. This project demonstrates turn-based logic, board management, and win-condition checks — making it an excellent example of problem-solving and interactive programming.

Key Features:
🎮 Two-player mode (Player X vs Player O)
🧠 Intelligent game board representation using lists
✅ Input validation to prevent overwriting moves
🏁 Win, lose, or draw detection logic
🔁 Option to restart the game

Technologies Used:
Python (Core Game Logic)
Lists and Functions (to manage board and game flow)
Conditional Statements (to check win/draw)
Loops and Input Handling (for continuous play)

📱 Responsive design for different screen sizes

Technologies Used:
HTML – To structure the web page elements
CSS – For styling, layout, flexbox/grid usage, and responsiveness
